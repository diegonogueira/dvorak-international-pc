# OSX Install

## Method 1 - Automatic

Use the `Dvorak International - PC.dmg`.

## Method 2 - Manual

1 - Put the `Dvorak International - PC.bundle` on `/Library/Keyboard Layout/` folder.

2 - Open `Preferences` > `Keyboard` > `Input Source`.

3 - Click on `+` and choose the `Dvorak International - PC`.
